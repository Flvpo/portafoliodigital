Calculadora Avanzada en C

Se desarrolló una calculadora avanzada en C con funciones matemáticas y manejo de errores. Este proyecto requirió habilidades en programación en C, control de flujo y gestión de datos de entrada/salida. La calculadora ofrece operaciones como suma, resta, multiplicación, división, potencia, raíz cuadrada, seno y coseno, con una interfaz de usuario interactiva. Es una aplicación versátil que demuestra competencia en programación y resolución de problemas.

Juego de Batalla Naval Python

El juego de Batalla Naval en Python emplea listas, control de flujo, entrada del usuario y generación aleatoria para que el jugador adivine la ubicación de los barcos de la máquina en una cuadrícula 5x5. Se utilizan funciones personalizadas para crear y mostrar la cuadrícula, además de gestionar la lógica del juego. Los mensajes guían al jugador, y las variables llevan un registro de los intentos y el estado del juego. La victoria se logra al hundir un barco de la máquina. Este proyecto ofrece una introducción práctica a conceptos de programación como bucles, estructuras de datos y lógica de juego en Python.

Identificacion de USB en Python

Se utilizó Python y la biblioteca pywinusb para enumerar dispositivos USB en Windows. El código importa pywinusb.hid, define una función enumerar_dispositivos_usb() y utiliza find_all_hid_devices() para listar dispositivos HID USB. Luego, se muestra información como el ID del vendedor, el ID del producto y el nombre del dispositivo. Las habilidades clave incluyen programación en Python, el uso de bibliotecas externas, la iteración y la comprensión de cómo interactuar con dispositivos USB en un entorno Windows.
